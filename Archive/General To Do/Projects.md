# Current Projects


### ResearchAlly Messaging
This project will add messaging functionality to Research Ally. This will allow researchers and participants to communicate securely during the recruitment phase of a study.

### New Posting Tool
Begin experimenting with a new Posting Tool which can be more easily modified, perhaps declaratively, like a SproutForm.
