## Usage Guidelines

- **Keep personal separate from work** - No work content in this vault
- **Use descriptive file names** - Easy to find what you need
- **Link related documents** - Use Obsidian `[[wiki links]]` syntax
- **Organize by topic** - Place files in appropriate directories
